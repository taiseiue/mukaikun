## これは何
- Moodleで掲示されているPDFなどのファイルを、ダウンロードせずにブラウザ内で読み込むChrome拡張機能です。
- Moodle以外のサイトでも、パターンを書くことでブラウザ内で読み込めます。
- 初めましての人は、まず下の設定欄にMoodleのUrlを入力しよう。
